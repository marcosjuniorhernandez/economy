# Teoría de la Producción

La función de producción **Cobb-Douglas** se define como:

$$Y = A \cdot L^\beta \cdot K^\alpha$$

Donde:
* $Y$ es el producto total.
* $L$ es el trabajo (*Labor*).
* $K$ es el capital.
* $\alpha$ y $\beta$ son las elasticidades del producto.

Si $\alpha + \beta = 1$, tenemos rendimientos constantes a escala.
