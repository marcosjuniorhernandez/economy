# Installation

Dependiendo de tu caso de uso, existen diferentes formas recomendadas de instalar **Oikos**.

Si solo deseas explorar la librería o ejecutar ejemplos simples, una instalación básica con `pip` es suficiente.  
Si planeas trabajar con modelos económicos de forma recurrente (docencia, investigación o desarrollo), se recomienda usar un entorno aislado.

---

## Requisitos

- Python 3.10 o superior
- Se recomienda el uso de entornos virtuales

---

## (Recomendado) Instalación con pip

La forma recomendada de instalar **Oikos** es mediante el gestor de paquetes de Python `pip`.

Si ya tienes un entorno de Python configurado, ejecuta:

```bash
pip install oikos
```
